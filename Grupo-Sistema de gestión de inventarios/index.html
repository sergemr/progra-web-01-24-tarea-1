Presentación Tema Trabajo Final

Curso Programación Web - I Cuatrimestre 2024

Breve Explicación:

El proyecto consiste en desarrollar un sistema de gestión de inventarios que permita a una organización mantener un control eficiente sobre sus activos, así como facilitar el proceso de préstamos y devoluciones. Además, incluirá un sistema de registro de usuarios para gestionar quiénes pueden solicitar préstamos y acceder al sistema.
