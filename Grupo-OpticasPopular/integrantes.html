Se usará Node.js para el backend, dentro de este se implementará un sistema de usuario y contraseña , donde usuarios se podrán registrar y entrar con las credenciales que ellos creen para solicitar citas o tener acceso al historial medio de citas.
Ademas todo va a estar conectado a una base de datos mysql y en react vamos a trabajar el frontend de la pagina web para el usuario que va poder hacer su login o sign up.
